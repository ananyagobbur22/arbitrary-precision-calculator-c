# Arbitrary Precision Calculator (C)

## 📌 Overview
This project implements an arbitrary precision calculator that performs arithmetic operations on large numbers beyond standard data type limits. It uses doubly linked lists to store and manipulate digits dynamically.

## 🚀 Features
- Addition of large numbers
- Subtraction with borrow handling
- Multiplication (if implemented)
- Supports numbers of any size
- Dynamic memory-based computation

## 🛠️ Technologies Used
- C Programming
- Doubly Linked Lists
- Dynamic Memory Allocation
- Pointer Manipulation

## ⚙️ How It Works
- Each digit is stored as a node in a doubly linked list
- Numbers are processed digit-by-digit
- Carry and borrow handled across nodes
- Results constructed dynamically
